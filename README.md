# 运营服务2.1.8接口



## 1.续联数据统计查询

请求url：/static/trace/getTraceDataStatis

请求参数：

| 字段说明           | 字段名         | 类型   | 备注                                |
| ------------------ | -------------- | ------ | ----------------------------------- |
| 汇总展示、按天展示 | showFlag       | int    | 汇总1  按天2                        |
| 开始日期           | startTime      | string | yyyy-MM-dd                          |
| 结束日期           | entTime        | string | yyyy-MM-dd                          |
| 机构code           | code           | string | 多选时code以逗号分隔，如301,302,303 |
| 提交者姓名         | commitNameOrId | string |                                     |
|                    | page           |        |                                     |
|                    | rows           |        |                                     |



汇总返回：

| 字段说明     | 字段名           | 类型   | 备注 |
| ------------ | ---------------- | ------ | ---- |
| 提交者ID     | commitId         | string |      |
| 提交者名称   | realName         | string |      |
| 提交者角色   | userType         | string |      |
| 所属三级机构 | groupName        | string |      |
| 所属二级机构 | parentName       | string |      |
| 请求数量     | requestNum       | int    |      |
| 已续回       | traceSuccNum     | int    |      |
| 续联中       | traceIngNum      | int    |      |
| 未续回       | traceNotNum      | int    |      |
| 处理超时     | traceTimeoutNum  | int    |      |
| 续回率       | traceSuccPercent | string |      |
| 已拨打       | callNum          | int    |      |
| 拨打率       | callPercent      | string |      |
| 已接通       | talkNum          | int    |      |
| 接通率       | talkPercent      | string |      |

按天返回

增加 提交时间字段(string)：commitTime

## 2.续联数据统计导出

请求url：/static/trace/exportTraceDataStatis

请求参数：

| 字段说明           | 字段名         | 类型   | 备注                                |
| ------------------ | -------------- | ------ | ----------------------------------- |
| 汇总展示、按天展示 | showFlag       | int    | 汇总1  按天2                        |
| 开始日期           | startTime      | string | yyyy-MM-dd                          |
| 结束日期           | entTime        | string | yyyy-MM-dd                          |
| 机构code           | code           | string | 多选时code以逗号分隔，如301,302,303 |
| 提交者姓名         | commitNameOrId | string |                                     |

select commit_id commitId,real_name realName,user_type userType,group_code groupCode,group_name groupName,parent_name parentName,request_num requestNum,trace_succ_num traceSuccNum,trace_not_num traceNotNum,call_num callNum,talk_num talk_num  from statis_trace_ccod where commit_time BETWEEN '2020-09-17 00:00:00' and '2020-09-17 23:59:59' and ent_num = '2019042401' group by
